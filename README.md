# Codveda Data Science Internship Projects

This repository contains the complete set of data science projects completed during my internship at **Codveda Technology**. The project tasks are organized into three progressive levels, ranging from fundamental exploratory data analysis to deep learning neural networks.

---

## 📁 Repository Structure

### 🔹 Level 1: Basics & EDA
* **`preprocessing_script.py`**: Handles missing values, data cleaning, and feature scaling/normalization.
* **`eda_script.py`**: Conducts exploratory data analysis, descriptive statistics, and data distributions.
* **`EDA report.docx`**: A comprehensive, formal documentation of the insights gained during exploration.

### 🔹 Level 2: Intermediate Machine Learning
* **`house_prediction_model.py`**: A regression framework predicting real estate valuations.
* **`churn_knn_model.py`**: A K-Nearest Neighbors classification system identifying customer churn risks.
* **`segmentation_kmeans.py`**: Unsupervised customer segmentation utilizing K-Means clustering and the Elbow Method.

### 🔹 Level 3: Advanced Data Science & AI
* **`nlp_text_classification.py`**: Text classification system applying TF-IDF vectorization and classical classifiers.
* **`neural_networks.py`**: A Feed-Forward Neural Network architecture built using TensorFlow and Keras.

---

## 🛠️ Tech Stack & Libraries Used
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-Learn
* **Deep Learning:** TensorFlow, Keras
